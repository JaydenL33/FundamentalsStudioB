Welcome to SANITISE.media's documentation!
==========================================

The `SANITISE.media`_ project will consist of developing Virus Visualising Media, henceforth SANITISE (Showing and Naming Integral Tools in Slowing Epidemics). 

This project relevantly compares and visualises complex and dynamic data of the
SARS-COV-2 (a.k.a. COVID or Coronavirus) outbreak promptly. In conjunction with
efficiently and effectively delivering this data in an easily understandable
format.
